# ARCquick

ARCquick là một dự án trang web tĩnh đơn giản (single-page) dùng để demo hoặc làm khung nhanh cho các thử nghiệm front-end.

## Mục tiêu
- Cung cấp một trang `index.html` cơ bản để bắt đầu phát triển giao diện nhanh chóng.

## Cách chạy
- Mở file `index.html` trực tiếp bằng trình duyệt.
- Hoặc chạy server tĩnh trong thư mục dự án rồi truy cập tại `http://localhost:8000`:

```bash
python -m http.server 8000
```

## Cấu trúc thư mục
- `index.html` — file chính của dự án
